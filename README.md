# PrivateFileBox

#### 介绍
PrivateFileBox 私密文件柜 是一款专为 iOS 设计的本地文件隐私保护应用。核心功能包括：

1. 密码安全锁 ：启动时需输入密码验证，有效防止他人访问您的私密文件。
2. 多格式文件管理 ：支持图片、视频、PDF/Word/Excel 文档、网页等多种格式的分类存储与浏览。
3. 媒体预览播放 ：内置图片浏览器、专业视频播放器（BMPlayer）及文档 WebView 预览，无需第三方应用即可查看。
4. 相册导入导出 ：一键从系统相册导入照片与视频，支持选择导入画质，也可将文件导出回相册。
5. 局域网 HTTP 传输 ：通过内置 Web 服务器，在电脑浏览器即可无线上传下载文件，告别数据线。
6. 个性化设置 ：可随时修改访问密码、调整视频导入质量等偏好选项。
所有文件仅保存在应用沙盒本地，杜绝云端泄露风险。

PrivateFileBox is a local file privacy protection app designed exclusively for iOS. Key features include:

1. Password Security Lock : Requires password verification on launch to effectively prevent unauthorized access to your private files.
2. Multi-Format File Management : Supports categorized storage and browsing of various formats including images, videos, PDF/Word/Excel documents, and web pages.
3. Media Preview & Playback : Built-in image browser, professional video player (BMPlayer), and WebView document viewer enable viewing without third-party apps.
4. Album Import & Export : Import photos and videos from the system album in one tap, with selectable import quality. Files can also be exported back to the album.
5. LAN HTTP Transfer : Via the integrated web server, wirelessly upload and download files from any computer browser — no data cable needed.
6. Personalized Settings : Change access passwords, adjust video import quality, and customize other preferences anytime.
All files are stored exclusively in the app's local sandbox, eliminating the risk of cloud data leakage.
